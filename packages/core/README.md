# @node-core/doc-kit

A tool to generate API documentation for Node.js. See [the root README](../../README.md) for full usage details and examples.

## Structure

- `src/` - Source code (generators, parsers, utilities, threading)
- `bin/` - CLI entry point

## Scripts

```sh
# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Run the CLI
npm run run -- generate --help

# Watch mode
npm run watch
```
